Overview

A New Benchmark for Event-based Perception

The eAP dataset provides synchronized event camera streams and RGB images captured from real-world driving scenarios, paired with high-quality 3D bounding box annotations, object tracking, and time-to-collision (TTC) labels.

Data Modalities

Multi-sensor Synchronized Capture

Event Camera

High temporal resolution event streams capturing brightness changes at microsecond precision. Ideal for fast-moving objects and HDR scenes.

RGB Camera

Synchronized high-resolution RGB images providing rich texture and color information for visual perception tasks.

3D Annotations

Precise 3D bounding boxes with object tracking IDs, velocity estimation, and per-frame occlusion ratios in ego-vehicle coordinates.

Coming Soon

LiDAR

3D point cloud data with LiDAR-based annotations for comprehensive depth perception and cross-modal validation.

Annotations

Rich, Multi-dimensional Labels

3D Bounding Boxes

7-DoF cuboid annotations (x, y, z, l, w, h, yaw) in ego-vehicle coordinate system with front-camera projection.

Object Tracking

Consistent tracking IDs across frames, enabling trajectory analysis and motion pattern study.

Velocity & TTC

Per-object ego-relative speed vectors and time-to-collision (TTC) / headway TTC (HTTC) labels for safety-critical perception.

Occlusion & Calibration

Occlusion ratio per object, full intrinsic/extrinsic calibration, and temporal alignment between event and RGB sensors.
